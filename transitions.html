💡 How to use this page
  1. Read the relationship, not the word. The SAT tests category → word. You see two sentences, name the relationship, then pick the transition that signals it.
  2. One category at a time. For each panel below, read the definition, study the example, scan the comparison list, then read the Hard-question tiebreaker. Once you've got it, mark it understood.
  3. Don't memorise traps. Section 6 lists transitions that are never correct. The skill is recognising them as misdirections during the protocol, not adding them to a study deck.
📋 The Two-Sentence Protocol — every transitions question
  1. Read sentence 1 completely. Understand what it claims.
  2. Read sentence 2 completely. Understand what it claims.
  3. Ask: "What is the actual relationship?" — do not match keywords; analyse the logic.
  4. Name which of the 5 relationships this is (out loud or sub-vocally).
  5. Eliminate every option that signals the wrong relationship.
  6. Among the remaining options, choose the most precise match using the within-category tiebreakers.
