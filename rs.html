💡 How to use this page
  1. Every DSAT RS question follows the same template. "The student wants to [GOAL]. Which choice most effectively uses relevant information from the notes to accomplish this goal?" The variation is in the goal — and it falls into 8 recognisable types.
  2. Read the goal before the notes. If you read the notes first, you'll be drawn to the most interesting fact rather than the one the goal requires.
  3. The wrong answers are almost always factually true. That's the master trap (Accurate But Off-Task). Correctness alone is not enough — the choice must serve the stated goal.
  4. For each goal type, learn the prompt cues, the rule the correct answer must satisfy, and the distinctive trap shape. Mark a type understood once those three are automatic.
📋 The Goal-First Protocol — every RS question
  1. Find the "The student wants to…" sentence. Mark it.
  2. Name the goal type (one of the 8 below).
  3. Identify the scope: which subject, which attribute, which audience, which study aspect.
  4. State out loud what the correct answer must do. Do not read the choices yet.
  5. Now read each choice once. Eliminate any choice that doesn't serve the stated goal — even if it's factually correct.
  6. Among survivors, pick the most complete and most precisely targeted choice.

Every RS question has the same master trap, named in the bank's own explanations: Accurate But Off-Task.
