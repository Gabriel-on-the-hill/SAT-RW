Tutor Dashboard

Live view of every student's sessions, pulled from your published Google Sheet.

How do I get this URL?
In your Google Sheet: File → Share → Publish to web → choose the responses sheet → format Comma-separated values (.csv) → Publish → copy the link and paste it above. The dashboard reads it directly (Google serves published CSVs with CORS, so no Apps Script change is needed). It maps columns by header name — it expects headers like date, student, type, score, total, pct, avgSecs, blurCount, skillStats, and shows whatever it finds.
