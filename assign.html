Assignment Builder

A tutor tool. Fill in the week, pick skills and difficulty from the buttons, and it writes out the rows for you — paste them into your Plans sheet, or the code block into assignments.js. No typing skill names, no syntax to get wrong.

1. Sheet rows — paste into the "Plans" tab

Click in the box, select all, copy, then click the first empty cell under your headers in the Plans tab and paste.

2. Or — code block for assignments.js

Paste inside the HOMEWORK = { … } object in homework/assignments.js.
